# MyRide Bus Tracker for Home Assistant

Tracks school buses from the Tyler MyRide platform.

Features:
- Multiple students supported
- Live GPS map
- ETA sensors
- Stop override
- Automatic bus detection

Install through HACS:
1. Add this repository as a custom integration.
2. Restart Home Assistant.
3. Add integration "MyRide Bus".
4. Enter your MyRide username/password.
